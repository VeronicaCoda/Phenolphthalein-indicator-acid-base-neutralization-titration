## 该文件夹存放使用pytorch实现的代码版本
**model.py**： 是模型文件  
**train.py**： 是调用模型训练的文件    
**predict.py**： 是调用模型进行预测的文件  
**class_indices.json**： 是训练数据集对应的标签文件   
**data** 文件夹里分为了train和val两部分，分别用来训练和验证，只要将完成分类的照片分别放在“orange”或“yellow”子文件夹中即可
